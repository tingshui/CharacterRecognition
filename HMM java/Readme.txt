This package is used for preparing required dataset for HMM experiment on Matlab.
You just need to run characterReco.java

Input:
args[0]: letter.csv
args[1]: pro.csv
args[2]: location to place new transit.csv
args[3]: location to place new emiss.csv
args[4]: location to place new words.csv
args[5]: location to place new originSeq_mod.csv
args[6]: location to place new predSeq_mod.csv
